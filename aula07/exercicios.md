# Exercícios - Aula 07 - 07/04

## Exercícios – Strings com Laços de Repetição (Avançando)

Resolva os exercícios utilizando Strings e estruturas de repetição (`for` ou `while`).

---

### Exercício 1 – Contador de Consoantes

Crie um programa que:

1. Solicite uma frase ao usuário.  
2. Percorra a String caractere por caractere.  
3. Conte quantas **consoantes** existem na frase.  
4. Exiba o total.

📌 **Observação:**  
Ignore espaços, números e caracteres especiais.

---

### Exercício 2 – Remoção de Caracteres Específicos

Crie um programa que:

1. Solicite uma frase.  
2. Solicite um caractere para remoção.  
3. Percorra a frase e construa uma nova String sem esse caractere.  
4. Exiba o resultado.

### Exemplo

Entrada:
- Frase: `banana`  
- Caractere: `a`  

Saída:
- `bnn`

---

### Exercício 3 – Maior Sequência de Caracteres Iguais

Crie um programa que:

1. Solicite uma palavra ao usuário.  
2. Percorra a String e identifique a **maior sequência de caracteres iguais consecutivos**.  
3. Exiba o tamanho dessa sequência.

### Exemplo

Entrada: `aaabbccccd`  
Saída: `4` (sequência de 'c')

---

### Exercício 4 – Contagem de Palavras

Crie um programa que:

1. Solicite uma frase ao usuário.  
2. Percorra a String e conte quantas palavras existem.  
3. Considere que as palavras são separadas por espaços.

📌 **Desafio:**  
Tratar múltiplos espaços consecutivos corretamente.

---

## Desafio Extra

- Resolver sem utilizar métodos prontos como `split()`  
- Tentar implementar usando apenas `charAt()` e laços
